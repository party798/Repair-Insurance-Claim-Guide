# Repair-Insurance-Claim-Guide

이 저장소는 **의공수리 보험 청구 및 자산 반영 프로세스**를 실무자들이 한눈에 파악하고 쉽게 따라 할 수 있도록 시각화한 인포그래픽 웹페이지 프로젝트입니다.

부품 청구부터 최종 자산대장 반영까지의 유기적인 13단계 워크플로우를 담고 있으며, **GitHub Pages**를 통해 호스팅하여 **노션(Notion)** 이나 사내 그룹웨어에 깔끔하게 임베드해 사용할 수 있도록 최적화되어 있습니다.

---

## 🚀 주요 특징 (Key Features)

- **역할 기반 스윔레인(Swimlane) 레이아웃:** 각 담당자, 팀장, 자산관리 담당자별 업무 범위를 색상별로 분리하여 시각적 직관성을 높였습니다.
- **AI(GPTs) 지침 연동:** 손망실보고서 작성을 지원하는 맞춤형 GPTs 링크가 포함되어 있어, 마우스 클릭 한 번으로 신속하게 이동할 수 있습니다.
- **100% 반응형 웹 디자인:** PC, 태블릿, 모바일 등 어떤 화면 환경에서도 레이아웃이 깨지지 않고 가독성이 유지됩니다.
- **임베드 최적화:** 하나의 HTML 파일로 모든 스타일이 완벽히 포함되어 있어 노션(Notion) 임베드나 웹 아카이브에 매우 용이합니다.

---

## 📂 파일 구성 (Directory Structure)

- `index.html`: 인포그래픽 전체 소스코드가 포함된 메인 웹페이지 파일 (HTML/CSS)
- `README.md`: 프로젝트 개요 및 배포 가이드 문서 (본 파일)

---

## 🛠️ 배포 및 노션 임베드 방법 (Deployment & Notion Embedding)

### 1. GitHub Pages를 활용한 웹 호스팅
1. 본 레포지토리의 `Settings` 탭으로 이동합니다.
2. 좌측 메뉴에서 `Pages`를 클릭합니다.
3. **Build and deployment** 섹션의 Source를 `Deploy from a branch`로 설정합니다.
4. Branch를 `main` (또는 코드가 있는 브라우저 브랜치) 및 `/root`로 선택한 후 `Save` 버튼을 누릅니다.
5. 수 분 내에 상단에 상시 접속 가능한 웹페이지 URL(`https://<username>.github.io/<repo-name>/`)이 생성됩니다.

### 2. 노션(Notion) 페이지에 임베드하기
1. 위에서 생성된 GitHub Pages의 웹 URL을 복사합니다.
2. 노션 페이지의 원하는 위치에서 `/임베드` 또는 `/embed` 명령어를 입력합니다.
3. 복사한 URL을 붙여넣고 엔터를 누릅니다.
4. 가로/세로 길이를 조절하여 인포그래픽이 전체 화면에 깔끔하게 나오도록 맞춥니다.

## 📋 핵심 업무 프로세스 요약 (13 Steps Workflow)

### **Phase 1. 사전 검토 (Review)**
- **01단계:** 부품 청구 및 결재 상신 (각 담당자)
- **02단계:** 부품청구 반려 - 보험청구 필요 시 (팀장)

### **Phase 2. 보험청구 진행 (Claim Process)**
- **03단계:** 부서 내 손망실보고서 작성 요청 및 보험청구 협조전 발송 (각 담당자)  
  *(💡 GPT 지침을 참조하여 문구 작성)*
- **04단계:** 보험청구 결재라인 결재 후 KIS 장비명을 자산명+`$`로 변경 (자산관리 담당자)
- **05단계:** 부품청구서 + 보험청구 협조전 첨부하여 부품 청구 결재 재상신 (각 담당자)

### **Phase 3. 현장 대응 및 심사 (Investigation)**
- **06단계:** 연 1회 총무팀에 리스트 메일 발송하여 진행사항 확인 (자산관리 담당자)
- **07단계:** 손해사정사가 요청하는 서류 작성 및 제출 (각 담당자)
- **08단계:** 손해사정사 현장 대응 - 사고경위 설명 및 장비 상태 확인 지원 (각 담당자)
- **09단계:** 추가 보완자료 요청 시 대응 - 사진, 로그, 사용기록 등 제출 (각 담당자)

### **Phase 4. 정산 및 사후 처리 (Settlement & Close)**
- **10단계:** 보험금 산정 결과 확인 및 이의 여부 검토 (각 담당자)
- **11단계:** 보험금 지급 진행 확인 및 완료 전자결재 작성 (각 담당자)
- **12단계:** 최종결재 (팀장)
- **13단계:** 자산대장 및 처리결과 최종 반영 (자산관리 담당자)
